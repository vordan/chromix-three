Chromix Three

Version 1.0.1

Developer tool for controlling Chrome from the command line.

What is this?

This extension connects to a local server and allows you to control Chrome tabs from command-line scripts or editor shortcuts.

Primary use case: Reload development server tabs instantly while coding.

Requirements

A local Node.js server must be installed and running for this extension to work.

The server listens on:

  • HTTP: localhost:8444
  • WebSocket: localhost:7444

Installation

1. Install the server from the project repository

2. Start the server: ./server-start.sh

3. This extension will automatically connect

Usage

Use command-line scripts to control Chrome:

  • Reload tabs matching URL pattern
  • List all open tabs
  • Close tabs by URL
  • Open new tabs
